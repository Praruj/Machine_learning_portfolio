## Machine Learning and Deep learning 
- It includes `Supervised Machine Learning` (Labeled Data) and `Unsupervised ML` (Unlabeled Data)

1. Supervised ML : Includes Regression and Classification.
    - Independent Variables (Features): Predictor
    - Dependent variables (Target): prediction

- Regression, Used when the target is continuous (any number within a range, like $250,000.75 or 23.7°C).,  [A continuous target variable (also called a continuous dependent variable) represents numerical values that can take any point within a range, rather than fixed categories].

- For classification, Used when the target is discrete/categorical (fixed options).
    - Binary Classification : Predicts one or two outcomes. Like Yes/no?. 
    - Multi class classification: Predicts from 3+ categories. like dog(0), cat(1), bird(2). 

**Algorithms in Supervised ML**
1. Linear regression
2. Ridge and Lasso
3. Logistic Regression
4. Adaboost
5. Random forest
6. Gradient Boost
7. Decision Tree
8. Xgboost
9. Naive Bayes 
10. KNN


2. Unsupervised ML: Clustering & Dimensionality Reduction

- Clustering: Grouping things together, where there is no dependent variable (no output features)  
- Dimensionality reduction: Lowering dimension. 

**Algorithms in Unsupervised Ml**
1. K Means
2. DBscan
3. K nearest Neighbout
4. PCA
5. LDA
6. Hierarchical Clustering
